# Tetris
A tetris game written in rust using the comfy game engine: https://github.com/darthdeus/comfy

![Screenshot](https://github.com/IvanDimovSIT/tetris/blob/master/screenshot.png)

# Controls:
A/Left - Move left

D/Right - Move right

S/Down - Move down

W/Up - Rotate right

Z - Rotate left

Space - Place

Esc - Pause/Unpause
